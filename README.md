# 🖥️ Sistema de Gestão - Java

Sistema desktop de cadastro e gerenciamento de clientes e produtos desenvolvido com Java, MySQL e JasperReports para fins de estudo em desenvolvimento de aplicações desktop.

O projeto permite login de usuários, cadastro de clientes e produtos, visualização de registros em tabelas e geração de relatórios em PDF, simulando um pequeno sistema de gestão.

🚀 Funcionalidades

✔ Login de usuário com validação no banco de dados  
✔ Cadastro de clientes  
✔ Cadastro de produtos  
✔ Listagem de clientes  
✔ Listagem de produtos  
✔ Visualização de detalhes em tabelas  
✔ Filtros de busca dinâmicos (nome, email, telefone, fabricante, etc.)  
✔ Geração de relatórios em PDF

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com objetivo educacional, para praticar conceitos importantes de programação desktop, como:

conexão com banco de dados utilizando JDBC
uso de classes e orientação a objetos em Java
aplicação do padrão DAO (Data Access Object)
manipulação de dados com SQL
controle de autenticação de usuários
criação de interfaces gráficas com Swing
geração de relatórios com JasperReports

## 🧠 Conceitos utilizados no projeto

Durante o desenvolvimento foram utilizados alguns conceitos importantes:

padrão DAO para acesso ao banco de dados
separação entre model, view e dao
consultas dinâmicas ao banco de dados
uso de DefaultTableModel para tabelas Swing
eventos de teclado para filtros em tempo real
controle de login de usuários
geração de relatórios com JasperReports
