🖥️ Sistema de Gestão - Java Swing

Sistema desktop de cadastro e gerenciamento de clientes e produtos desenvolvido com Java, MySQL e JasperReports para fins de estudo em desenvolvimento de aplicações desktop.

O projeto permite realizar login de usuários, cadastrar clientes e produtos, visualizar registros em tabelas e gerar relatórios em PDF.

🚀 Funcionalidades

✔ Login de usuário com validação no banco de dados
✔ Cadastro de clientes
✔ Cadastro de produtos
✔ Listagem e visualização de clientes
✔ Listagem e visualização de produtos
✔ Filtros de busca dinâmicos (nome, email, telefone, fabricante, etc.)
✔ Geração de relatórios em PDF

🛠 Tecnologias utilizadas
Front-end

☕ Java Swing (Interface gráfica)

Back-end

☕ Java (POO + DAO Pattern)
🧩 JDBC

Banco de Dados

🗄️ MySQL

Relatórios

📊 JasperReports

🎯 Objetivo do projeto

Este projeto foi desenvolvido com objetivo educacional, para praticar conceitos importantes de desenvolvimento desktop, como:

conexão com banco de dados utilizando JDBC
padrão DAO (Data Access Object)
programação orientada a objetos em Java
manipulação de dados com SQL
criação de interfaces gráficas com Swing
geração de relatórios com JasperReports
organização em camadas (Model / View / DAO)
🧠 Conceitos utilizados no projeto

Durante o desenvolvimento foram aplicados alguns conceitos importantes:

padrão DAO para acesso ao banco de dados
separação entre model, view e dao
consultas dinâmicas no MySQL com filtros (LIKE)
uso de DefaultTableModel para manipulação de tabelas no Swing
eventos de teclado para busca em tempo real
controle de autenticação de usuários
geração de relatórios com JasperReports
